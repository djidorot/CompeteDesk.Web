# CompeteDesk

CompeteDesk is a strategic decision-support web application designed to help individuals and teams plan, analyze, and execute competitive strategies using structured frameworks and AI-assisted insights.

## Favorite Quote

> “The best way to predict the future is to invent it.” — Alan Kay

## Features

- Workspaces for organizing strategic initiatives
- Strategies and Actions with AI-assisted playbooks
- War Room for competitive intelligence and planning
- Business and Website Analysis reports
- Habit tracking with AI suggestions
- AI decision trace and audit history

## Tech Stack

- ASP.NET Core MVC (.NET 9)
- Entity Framework Core + SQLite
- ASP.NET Identity + Google OAuth
- OpenAI Chat Completions API
- Razor Views, CSS, Vanilla JavaScript

## Running the Project

1. Install .NET 9 SDK
2. Run:
   dotnet run
3. Open:
   http://localhost:5019

## Configuration

Update appsettings.json for:

- OpenAI API key
- Google OAuth credentials (optional)
